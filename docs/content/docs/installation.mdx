## Flutter app

Add the widget package:

```bash
flutter pub add humation_flutter
```

[`humation_flutter` on pub.dev](https://pub.dev/packages/humation_flutter).

That is everything. It bundles the default artwork pack (86 parts), so there is
nothing else to download or configure.

Import it:

```dart
import 'package:humation_flutter/humation_flutter.dart';
```

## Dart only, no Flutter

To generate avatars as SVG strings on a server or in a CLI, add the engine and an
artwork pack instead:

```bash
dart pub add humation humation_assets_humation_1
```

[`humation`](https://pub.dev/packages/humation) and
[`humation_assets_humation_1`](https://pub.dev/packages/humation_assets_humation_1)
on pub.dev.

See [Packages](/docs/packages) for how the three packages fit together.

## Prewarm (optional)

The artwork decodes once, the first time you show an avatar. Move that work off
the first frame by calling `prewarm` in `main`:

```dart
void main() {
  Humation.prewarm();
  runApp(const MyApp());
}
```
